Design summary

  • Desktop: 60/40 split — left hero panel (gradient + illustration + branding + feature pills), right auth panel (welcome text + buttons)
  • Mobile: Curved hero header (~50% height) with illustration background, feature pills below, buttons pinned at bottom
  • Hero panel: Reuses AuthHeroPanel pattern and CurvedBottomClipper from existing auth widgets
  • Illustration: Placeholder image slot — user will provide a custom welcome illustration later
  • Feature pills: 3 compact highlights (formats, tracking, organization) shown on both layouts
  • E-ink: Falls back to simple centered layout with no gradient/illustration (existing e-ink theme handles this)
