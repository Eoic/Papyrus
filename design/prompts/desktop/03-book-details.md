### Filter Components

**Radio Group (Reading Status):**
- Single select
- 24px radio buttons
- 40px row height

**Checkbox Group (Format):**
- Multi-select
- 24px checkboxes
- 40px row height

**Rating Slider:**
- Range: 1-5 stars
- Slider with star display
- "X stars and above" text

**Date Range:**
- Two date pickers
- "to" separator
- Calendar popup on click

**Multi-select Dropdown:**
- Chip display for selections
- Dropdown with checkboxes

**Apply Button:**
- Full width
- Primary filled
- Shows count: "Apply 5 Filters"
